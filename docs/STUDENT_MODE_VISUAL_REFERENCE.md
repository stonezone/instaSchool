## Color Palette

### Primary Colors
- **Success Green**: `#10b981` (Completed items, correct answers)
- **Primary Blue**: `#3b82f6` (Current/active items)
- **Warning Orange**: `#f59e0b` (Quizzes, important actions)
- **Error Red**: `#ef4444` (Incorrect answers)
- **Locked Gray**: `#9ca3af` (Locked units)

- **Background**: `#ffffff` (Light mode), `#1f2937` (Dark mode)

## Responsive Breakpoints

### Desktop (> 1024px)
- Sidebar: 320px fixed width
- Main content: Remaining width
- 3-column layouts where appropriate

### Tablet (768px - 1024px)
- Sidebar: 280px fixed width
- Main content: Remaining width
- 2-column layouts

### Mobile (< 768px)
- Sidebar: Collapsible drawer
- Main content: Full width
- Single column layouts
- Stacked navigation buttons

## Animation Specifications

### Level Up
- Duration: 1.5s
- `st.balloons()` triggered
- Success message with prominent display
- XP progress bar updates smoothly

### XP Gain
- Duration: 0.5s
- Success toast/message
- Progress bar increments
- Number counter animates

### Quiz Feedback
- Duration: 0.3s
- Immediate color change (green/red)
- Icon appears (âœ…/âŒ)
- Message slides in

### Section Transition
- Duration: 0.2s
- Fade out current content
- Fade in new content
- Progress bar updates

---

## Accessibility Features

### Keyboard Navigation
- Tab through all interactive elements
- Enter/Space to activate buttons
- Arrow keys for radio button selection
- Escape to close expanded sections

### Screen Reader Support
- All images have alt text
- Progress bars have aria-labels
- Questions have proper heading structure
- Status changes announced

### Visual Accessibility
- Minimum font size: 16px
- Color contrast ratio: 4.5:1 minimum
- Focus indicators on all interactive elements
- Error messages not dependent on color alone
